# 总 PRD：本地照片整理器

这个文件是当前产品的完整事实。已经上线的能力、当前业务规则、页面和限制都要反映在这里。

## 产品定位

本地照片整理器帮助用户在局域网或本机环境中整理照片，优先保证本地可控、低学习成本和可恢复。

## 当前已上线能力

### 本地目录索引

用户可以选择一个本地照片目录，系统会读取图片文件并建立基础索引。

已上线规则：

- 支持常见图片格式。
- 不上传用户照片。
- 索引失败时保留错误记录。

来源需求：

- `docs/prd/requests/2026/Q2/completed/local-index-v1.md`

## 当前计划中的能力

- 导入文件夹时支持增量扫描。
- 检测疑似重复照片。

## 当前非目标

- 不做云同步。
- 不做多人协作。
- 不做公开分享相册。

## 页面和模块

- 首页：显示已接入的照片目录。
- 导入页：选择目录、查看扫描进度和错误。
- 照片列表：浏览索引后的照片。

## 上线后反写规则

需求完成后必须更新本文件：

- 如果新增了用户可见能力，写入“当前已上线能力”。
- 如果改变了业务规则，更新对应规则。
- 如果删除或推迟能力，更新“当前非目标”或限制说明。
