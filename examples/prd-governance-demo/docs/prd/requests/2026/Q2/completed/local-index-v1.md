## 背景

用户需要先把本地照片目录接入系统，后续才能做整理、去重和筛选。

## 目标

- 选择一个本地目录。
- 扫描目录中的图片文件。
- 建立基础索引。
- 记录扫描失败的文件。

## 非目标

- 不做云上传。
- 不做图片内容分析。

## 验收标准

- 用户能选择本地目录。
- 系统能展示扫描成功数量和失败数量。
- 失败文件有可追踪错误。
