# 疑似重复照片检测

status: planned
created_at: 2026-07-01
target_stage: 产品需求
related_workstream:

## 背景

用户整理照片时经常遇到重复或近似重复图片，需要先识别再决定是否删除。

## 目标

- 找出疑似重复照片。
- 给出可解释的重复原因。
- 删除动作必须由用户确认。

## 非目标

- 不自动删除照片。
- 不上传图片做云端识别。

## 待澄清

- 是只做完全重复，还是做相似照片？
- 是否需要缩略图对比界面？
- 删除后是否进入回收站？
